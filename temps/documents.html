What is the FWALL?

Iptables: It is the default, command-based security firewall for the Linux operating system.It talks to Linux kernel and determines packet filtering rules. This firewall can block traffic passing through the ports that the services are running, it can route to another port.
Fwall: Fwall is a web interface for iptables.

How to use FWALL?

There are three types of chain, input, forward and output.
Input: This chain is used to control the behavior to be determined according to incoming connections. For example, if a user tries to connect with SSH to your computer or server, it tries to match the IP address and port number according to the rules in the iptables input chain.
Output: This chain is used for outbound links.
Forward: This chain is used to configure incoming connections that will not be delivered to local devices. For example, if you try to ping google.com. Iptables; it will check the output chain to make a decision to reject or accept the connection to and google.com.
